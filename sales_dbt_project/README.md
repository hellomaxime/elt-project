### Commands:
Install __dbt__ :  
`pip install dbt-postgres`  

Check :  
`dbt --version`  

Initialize project :  
`dbt init`  

Configure profile :  
`~/.dbt/profiles.yml`  

Verification :  
go to project directory  
`dbt debug` should return "All checks passed!"  

Run dbt project :  
`dbt run`
